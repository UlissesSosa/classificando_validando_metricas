#  Classificação com Machine Learning

## 📊 Sobre o Projeto
Neste curso, vamos criar um projeto de classificação com Machine Learning para identificar clientes inadimplentes em uma empresa de empréstimo de automóveis.


## 🎯 Objetivos de Aprendizagem

### 1. Validação de Modelos ML
- **Hold-Out**: Divisão treino/teste (70/30)
- **Validação Cruzada**: Cross-validation com 5 folds

### 2. Métricas de Desempenho
- Acurácia, Precisão, Recall, F1-Score
- Matriz de Confusão
- Curva ROC e AUC

### 3. Balanceamento de Dados
- **Oversampling**: SMOTE (Synthetic Minority Over-sampling)
- **Undersampling**: NearMiss

### 4. Pipeline de Validação
- Pré-processamento automatizado
- Prevenção de data leakage
- Reprodutibilidade
